# autosize-textarea

> textarea：组件实现了根据**内容高度**自动调节**高度**。





## options配置项

> 无



## 方法

### autosize(jQuery/HTMLElement)

```javascript
define([
    'components/textarea/autosize-textarea'
], function (textarea) {
    // 调用
    textarea.autosize($('textarea'));
});
```



> 参数：
>
> el：`jQuery元素`或者`DOM`元素
>
> 类型：`jQuery`或者`HTMLElement`
